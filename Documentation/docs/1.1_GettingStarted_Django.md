### 1. Configure Virtual Environment and Install Django

1. Create Virtual Environment.
   
         python -m venv [env name].

2. Update PIP

        python -m pip install --upgrade pip

3. Activate venv.
    
        [env_name]\scripts\activate
        
4. Install Django.
        
        python -m pip install django

5. Start project to create framework/files.

        django-admin startproject [projectname] .

### 2. Build Database

    python manage.py migrate

### 3. Run the Web Server

    python manage.py runserver

### 4. Start the Web App

1. Open a new terminal (CTRL + SHIFT + `).
2. Navigate to venv/directory with Manage.py.
3. Activate venv.
4. Navigate to project directory.
5. Start the app to create the framework/files for the project.

        python manage.py startapp [app_name]

### 5. Creating Models

1. Define the model in models.py.
2. Activate the model by adding the model to settings.py.
3. Stage the model with makemigrations.

        python manage.py makemigrations

4. Push the model to db with migrate.

        python manage.py migrate

5. Register the model with admin site.

### 6. Establish Django Admin Site

1. Create Superuser.

        python manage.py createsuperuser
        
2. Register models with admin site.
        **admin.py**

        from django.contrib import admin

        # Register your models here.

        from .models import Topic

        admin.site.register(Topic)
